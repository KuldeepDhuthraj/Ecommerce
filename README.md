# 🛒 Ecommerce Website

A responsive frontend Ecommerce website built using HTML, CSS, and JavaScript. The project includes a modern UI with product listings, shopping cart functionality, user profile, and contact page.

## 🚀 Features

- 🏠 Home page
- 🛍️ Product listing page
- 🛒 Shopping cart
- 👤 User profile page
- 📞 Contact Us page
- 📱 Responsive design
- ⚡ Interactive JavaScript functionality

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/KuldeepDhuthraj/Ecommerce.git
```

2. Open the project folder.

3. Open `Home.html` in your web browser.
